Why you should use Quanta CSS?

Built Different - Built for Speed

Quanta CSS gives you the power to build, scale, and style effortlessly with a modern utility-first approach and flexible components.

Utility‑First

Design straight from your markup with atomic classes. Skip the extra CSS and move faster with clarity.

Component‑Ready

Use prebuilt UI components that are styled, consistent, and easy to customize for any project.

Theme‑Ready

Switch themes instantly with built-in CSS variables and data attributes — light, dark, or custom.

Performance‑First

No runtime. No bloat. Everything is purgeable — optimized for speed from the core up.

Get Started with Quanta CSS

Use Quanta CSS your way — via CDN, NPM, or direct build integration. Choose what fits your flow and ship fast. 

<div class="d-flex flex-col gap-2 p-6 bg-grey-100 dark-bg-slate-700 rounded-lg">
<div class="d-flex flex-row text-left max-h-fit bg-white dark-bg-slate-800 rounded-lg overflow-hidden">
  <div class="d-flex max-w-28 max-h-fit overflow-hidden">
    <img src="..." alt="Preview" style="max-height: 200px; object-fit: cover;">
  </div>
  <div class="p-4">
    <h3 class="text-black dark-text-white font-weight-bold font-size-6">Nike Air</h3>
    <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit. Fuga</p>
    <div class="mt-2 d-flex gap-4">
      <button class="quanta-btn quanta-btn-primary w-full">Buy Now</button>
      <button class="quanta-btn-icon quanta-btn-rounded-md quanta-btn-outline-secondary">
        <svg ... ></svg>
      </button>
    </div>
  </div>

CDN Link

Just drop it in your HTML.

<link 
  href=
  "https://cdn.jsdelivr.net/npm/quanta-css@1.0.5/dist/quanta.css" 
  rel=
  "stylesheet" 
/>

NPM Package

Install via your package manager.

npm install quanta-css

Build Tool

Import into your CSS pipeline manually.

@import "quanta-css/dist/quanta.css"; 
<button class="quanta-btn quanta-btn-danger max-h-fit">Danger</button>
<button class="quanta-btn quanta-btn-success max-h-fit">Success</button>
<button class="quanta-btn quanta-btn-primary max-h-fit">Primary</button>
<button class="quanta-btn quanta-btn-outline-info max-h-fit">Info</button>
<button class="quanta-btn quanta-btn-outline-secondary max-h-fit">Secondary</button>
