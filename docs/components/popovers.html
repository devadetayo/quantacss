Interactions

Popover

Wrap any element in .quanta-popover-wrapper, then toggle a .quanta-popover with [data-show="true"]. 

<div class="quanta-popover-wrapper">
  <button id="trigger" class="quanta-btn">Click me</button>
  <div class="quanta-popover quanta-popover-top" data-show="true">
    Hello, I’m a popover!
  </div>
</div>
Hello, I’m a popover!
